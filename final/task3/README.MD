Yet another Tic Tac Toe game

Features:
* Default mode, game runs till one line is completed (set by default)
* Time mode, configurable overall time (20 secs. by default) to complete as much lines as possible
* Configurable board size (8x8 set by default)
* Configurable (through the source file) minimum required length of sequence (of equal characters) to win (5 by default)
* `Start again` button

(!) Max board size is limited to 20 for perfomance reasons.
